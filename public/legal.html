PRIVACY POLICY

This application uses some user related data from spotify.

All these data are accessed with user permission.

1- user-read-private= This scope used to access user's personal information. Spotaste uses this scope to access user's name and profie photo of their Spotify account.

2- user-read-email= Just like the first scope, this scope is also required to access user's personal information. First two scopes are used to access user's name and profile photo.

3- user-top-read= This scope used to access user's top tracks and artists in a specific time range. Spotaste uses this data to compare user's taste of music with other user.

4- user-follow-modify= This scope used to modify user's followers (follow or unfollow any user or artist). Spotaste uses this scope to let the user follow a specific Spotify user.
